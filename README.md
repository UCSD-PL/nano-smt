nano-smt
========

"Toy" SMT Solver for Educational Purposes.

This will support:

- Propositional Logic     /DPLL
- EUF                     /CC
- Difference Constraints  /Bellman-Ford

Over time, we may add:

- Fancy SAT heuristics 
  - non-chronological backtrack, etc.

- Linear Arithmetic / Simplex
